## Asset Reference

| File | Usage | Notes |
|---|---|---|
| `assets/gruve-logo.svg` | Nav + footer (light themes) | Fill `#062D1F` — use as-is on white/canvas |
| `assets/gruve-logo.svg` | Nav + footer (dark themes) | Apply `filter: brightness(0) invert(1)` to make white |
| `assets/icon.svg` | Nav + footer (index.html) | Fill already `#ffffff` — use directly on dark bg |
| `assets/gruve-spark-banner-1.webp` | Hero background | Dark image, green light streaks — used in v2 and graphite |
| `assets/category-icons/*.svg` | Dept + industry cards | Fill `#21a55e` — display natively on white/canvas backgrounds |
